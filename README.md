A simple C++ 2d(for now) Physics Engine, for learning purposes

Currently supports:
1. Basic Vector arithmetics
2. Calculation of change in a Point/Particle's x,y position based on Gravity
3. Calculation of change in a RigidBody's x,y position and angle based on Force
4. Checking for intersections between polygons
